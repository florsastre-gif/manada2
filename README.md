MANADA — App B (Match Engine)

App B implementa el motor de compatibilidad entre adoptante y perro en adopción.

El objetivo es demostrar un sistema de matching explicable basado en reglas y scoring, priorizando bienestar y ajuste real entre perfil humano y perfil del perro.

Enfoque

El cálculo de compatibilidad se compone de:

Filtros duros
Condiciones que bloquean el match (ej. incompatibilidad con gatos, falta de experiencia requerida).

Scoring blando
Ajuste por energía, experiencia, convivencia con niños u otros perros y condiciones del hogar.

Resultado explicable
El sistema devuelve:

Score numérico (0–100)

Nivel de compatibilidad

Factores a favor

Riesgos o advertencias

Datos

Para esta versión se utiliza un dataset de prueba (seed data) integrado en la app con perfiles estructurados que simulan casos reales.

No hay conexión a base externa en esta etapa.

Alcance del MVP

Perfil manual del adoptante

Evaluación de compatibilidad en tiempo real

Sistema de “likes”

Exportación de resultados en JSON

Objetivo académico

Demostrar:

Diseño de reglas de decisión

Separación entre filtros duros y scoring

Explicabilidad del modelo

UX enfocada en responsabilidad, no en engagement
